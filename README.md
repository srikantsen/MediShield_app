## 📸 About

MediShield is a responsive healthcare plan quotation website that helps users find the best health insurance plan based on their personal profile. Users answer a few simple questions and get personalised plan recommendations with dynamic premium calculations.

---

## ✨ Features

- 🧾 **3-Step Quote Flow** — Profile → Plan Selection → Add-ons
- 🧮 **Dynamic Premium Calculator** — Prices adjust based on age, family size, conditions & coverage
- 📋 **3 Plan Tiers** — Essential Care, Shield Plus, Platinum 360°
- ➕ **6 Optional Add-ons** — Maternity, OPD, Dental, Wellness, Global Cover, Accident Guard
- 🛒 **Cart System** — Add multiple quotes, view monthly & annual totals
- 🔔 **Toast Notifications** — Smooth feedback on every action
- 📱 **Fully Responsive** — Works on mobile, tablet & desktop
